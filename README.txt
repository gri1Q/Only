Изменения адреса локальног сервера для работы Google reCaptcha



1)Нажмите на кнопку "Пуск" и введите "Блокнот" в поле поиска.

2)Найдите блокнот в результате поиска, нажмите правой кнопкой мыши на иконку и выберите "Запустить от имени администратора".

3)Если система запросит подтверждение, нажмите "Да".

4)В открывшемся блокноте выберите "Файл" -> "Открыть" и найдите файл hosts по пути C:\Windows\System32\drivers\etc\hosts.

5)Добавьте запись в конец файла

# Custom Hosts
127.0.0.1 a.fake.domain-name

и сохраните изменения.

Запуск проекта


1)Сделайте дамп базы данных находящийся в папке dump
2)В папке class файл DB.php укажите свой хост, имя пользователя, пароль, назвадие базы данных
3) Включите локальный сервер
4) Запустите проект по этому адресу    http://a.fake.domain-name/
